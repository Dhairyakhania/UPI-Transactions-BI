# UPI Transactions Analysis Dashboard (2024)

## Overview
This project is an interactive **UPI Transactions Analysis Dashboard** developed using **Power BI** to analyze transaction patterns, balances, and user behavior for the year **2024**. The dashboard utilizes advanced Power BI features such as **DAX**, **slicers**, **bookmarks**, and multiple visualizations to enable dynamic and insightful analysis.

---

## Objectives
- Analyze UPI transaction trends and account balances over time  
- Segment users based on age groups using DAX logic  
- Provide city-wise, currency-wise, and month-wise transaction insights  
- Enable interactive exploration through slicers and bookmarks  

---

## Key Features

### User Segmentation using DAX
- Implemented a DAX-based age group classification:
  - **A1** – Young users  
  - **A2** – Middle-aged users  
  - **A3** – Senior users  
- Allows analysis of transactions and balances by age group.

### Interactive Slicers & Synced Slicers
- Added multiple slicers for:
  - Bank Name Sent
  - Bank Name Received
  - City
  - Device Type
  - Gender
  - Age Groups
  - Merchant Names
  - Payment Method
  - Purpose
  - Transaction Type
- **Slicer synchronization enabled across two pages**, ensuring consistent filtering and a unified user experience while navigating between report pages.

### Visualizations with Bookmarks
- **Line Chart**:
  - Displays trends in balance and transaction values over time.
- **Column Chart**:
  - Compares transactions and balances across selected dimensions.
- **Bookmarks**:
  - Used to switch between different visual states for better user interaction and cleaner UI.

### Matrix Analysis Page
- A separate page featuring a **Matrix Visual** with:
  - Transaction Amount  
  - Remaining Balance  
- Dimensions included:
  - City  
  - Currency  
  - Month  
- Provides detailed, granular financial insights.
